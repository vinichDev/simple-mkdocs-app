# Учебный сайт на MkDocs с кастомной темой

Этот сайт демонстрирует:

- собственную тему на **HTML + CSS + JS**;
- сборку статики через **PostCSS** (автопрефиксы + минификация);
- валидацию и минификацию HTML на этапе CI;
- деплой на **GitHub Pages**.

## Что изменилось в теме

- Кастомные **header** и **footer**.
- Домашняя страница со стилизованным блоком-карточкой.
- Улучшенная типографика: читаемые заголовки, подсветка ссылок и кода.
